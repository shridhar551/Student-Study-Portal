# Student Study Portal 🎓

A modern, lightweight, and responsive Single-Page Application (SPA) web portal designed to help students track their academic progress, manage productivity, and stay organized. Built using pure front-end web technologies.

Live features include an automated productivity dashboard, persistent notes, dynamic to-do listings, calculated lecture attendance metrics, and a structural Pomodoro study clock block.

---

## 🚀 Core Features

*   **Productivity Dashboard:** Real-time updates showcasing saved notes count, remaining unresolved to-do items, and active tracking sync metrics.
*   **Notes Management:** A scratchpad workspace allowing students to record quick ideas, lecture overviews, or important formulas.
*   **Dynamic To-Do Lists:** Interactive list enabling creation, completion toggle, and swift deletion of personal study goals.
*   **Attendance Tracker:** Visual subject matrices that log present/absent classes and instantly calculate and display your overall attendance percentage.
*   **Pomodoro Timer:** Built-in focus intervals preset to standard 25-minute study intervals, 5-minute short breaks, and 15-minute extended rest frames.

---

## 🛠️ Built With

*   **HTML5:** Structured semantic markup layout.
*   **CSS3:** Clean, customized modern styling layout leveraging dynamic modern CSS root custom variables.
*   **JavaScript (ES6):** Client-side application core logic and structural state updates.
*   **FontAwesome:** Modern visual component layout interface symbols and action vector icons.
*   **Web Storage API (`localStorage`):** Retains database entries securely in your browser so variables are preserved across tab reloads without structural database servers.
